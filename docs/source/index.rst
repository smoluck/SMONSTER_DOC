Welcome to SMONSTER's documentation!
===================================

**SMONSTER** is a Kit compilation for Modo

This page is the Documentation that refer to it.
You can find the Kit page on  `SMONSTER V2 Kit for Modo 14.0 to 15.X <https://smoluck.gumroad.com/l/smonster_v2>`_
It offers *simple* and *intuitive* Kits to bring you a quicker workflow of creation within Modo and other tools like RizomUV, Marmoset Toolnbag or even Pixaflux.

Check out the :doc:`Basics` section for further information, including
how to :ref:`install` or :ref:`update` it.

.. note::

   This Documentation is under active development.
