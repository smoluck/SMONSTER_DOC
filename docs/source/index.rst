Welcome to SMONSTER's documentation!
===================================

**SMONSTER** is a Kit compilation for Modo

This page is the Documentation that refer to it.
You can find the Kit page on  `SMONSTER V2 Kit for Modo 14.0 to 15.X <https://smoluck.gumroad.com/l/smonster_v2>`_
and offers a *simple* and *intuitive* Kits to gbring you a quicker workflow of creation within Modo.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
